## mechanical-mastery-plus - r1.0.9f -> r1.1.0

## Major changes
Added alternative automation setups: Create is not the only way to process brass rods, for example
Added Compact Machines, because Hyperbox causes chunks corruption when used with Pretty Pipes and I couldn't find out a solution.
Hyperbox will be removed in the next version.
Added Compact Crafting with custom recipes for Compact Machines and for some Mekanism casing items and AE2 controllers
Added Sky Guis to manage sky islands in a GUI
